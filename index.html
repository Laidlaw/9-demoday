ioiu | Pattern Library

wtf?

The ioui Pattern Library is a new tool for front end development. Its a style guide, a checklist and a colleciton of snippets.

(ui stands for user interface)
Each site we make looks unique, so its easy to assume that the problems we solve for each site is unique.
Here's the Library. It is a growing document that combines Inuit, barebones and Foundation. Unlike bootstrap or foundation, these code modules do not depend on a larger framework. 
						@mixin bevel($background, $corner: top-right, $size:10%) {
						  @if $corner == bottom-left {
						    $corner: 45deg;
						  } @else if $corner == bottom-right {
						    $corner: 135deg;
						  } @else if $corner == top-right {
						    $corner: 225deg;
						  } @else if $corner == top-left {
						    $corner: 315deg;
						  }

						  background: $background; /* fallback */
						  background:
						  -moz-linear-gradient($corner,  transparent $size, $background $size);
						  background:
						  -o-linear-gradient($corner,  transparent $size, $background $size);
						  background:
						  -webkit-linear-gradient($corner,  transparent $size, $background $size);
						}

Design Language

  • By building this library, designers and developers will have a design language.
  • We'll be able to use that langauge when we outsource projects.
  • This soccer site uses a full-background module with a vertically-centered module.
  • Modules are responsive so pages don't have to be.
